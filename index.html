Research

Developing Parallel Computation Architectures

Senior design project focusing on accelerating biologically-accurate neuron simulation using the Hodgkin-Huxley model. Designed and implemented an algorithm for partitioning the work for multi-neuron simulation across several machines. Designed an algorithm to run a simulation using the Hodgkin-Huxley model on the GPU using NVIDIA CUDA.
